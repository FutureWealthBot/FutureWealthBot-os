# FUTUREWEALTHBOT OS

Core runtime skeleton repository for FUTUREWEALTHBOT OS.

## Current scope

This repository currently contains architecture-first scaffolding for:
- API Gateway
- Agent Runtime Engine
- Strategy Engine
- Automation Engine
- Shared types
- execution planning

## Constraints

This repository intentionally avoids business logic at this stage.

Current work is limited to:
- interfaces
- abstract classes
- routing layers
- type definitions
- migration and execution documentation

## Guiding principle

Consolidate first.
Implement features later.
